## Общие

### Глобальные изменения

1. Переписан принцип проведения документов с ручным распределением

2. При проведении документов, если был отключен функционал очистки НДС и имеется ручное распределение с указанным НДС, то в регистрах он будет очищен

### Отчет ОПиУ

1. В варианте отчета P&L: План-Факт добавлен отбор по версии плана

### Список «Деньги»

1. Исправлена ошибка при использовании произвольных колонок

2. Добавлен отбор по физическому лицу

### Кошелек

1. При создании операции по кошельку на основании управленческой ведомости, если заполнить физическое лицо, то движения по регистрам пройдут только по указанному физическому лицу

### Проекты

1. В проекты добавлена вкладка «Файлы»

2. Для проекта можно прикрепить несколько файлов. Все файлы хранятся в таблице с полями «Файл» и «Описание»

3. Во вкладке «Файлы» добавлено отображение всех присоединенных файлов документов, отраженных на вкладке «Документы» проекта

### Документ Отражение зарплаты (управленческий)

1. Добавлена группировка по сотруднику в разрезе статей

2. Добавлена сортировка по ФИО

3. Добавлено распределение суммы строки при группировке по сотруднику

4. Добавлена загрузка данных документа из табличного документа (напр. из Excel)

5. Добавлена таблица дополнительных начислений. Данные из нее уходят только в регистр Расходы

6. Для таблицы дополнительных начислений также реализована группировка, сортировка, распределение

7. При создании документа на основании другого, автоматически удаляются строки с не заполненной статьей ДДС

8. Добавлена таблица Итого. В ней отображаются сгруппированные данные по таблицам документа

### Автоправила ДДС

1. В статьях распределения добавлено отображение знака %

### Контроль лимитов расходов

1. В платежном календаре появился дашборд, который отражает данные Плана и Факта

2. В платежном поручении появился блок, который отражает данные Плана и Факта

## Общие

### Правила ДДС и ОПиУ

1. Добавлена возможность скопировать правило

2. Добавлены периоды действия правила

   1. Если на текущий период действуют два и более правила, то будет выбрано правило с наиболее подходящим (наименьшим) периодом действия

3. Добавлен поиск по правилам

### Проекты

1. Добавлен функционал дополнительных реквизитов

   1. Для его успешной работы необходимо запустить обработку «Настойки - Заполнить доп реквизиты и сведения»

### Система обновлений

1. Добавлено оповещение о доступном обновлении

   1. Оповещение выводится только пользователям с правами «Администратор»

   2. Оповещение выводится при запуске 1С, а также каждые 3 часа

   3. При нажатии оповещения происходит автоматический переход на форму получения обновления

2. На форму обновления добавлена плашка, в которой указан текущий релиз

3. Добавлена возможность указать в качестве описания релиза PDF документ

4. Добавлена возможность публикации в Яндекс.Диск единого обновления для нескольких одинаковых конфигураций (например УТ / КА / ERP)

## БП

1. В подсистему добавлен документ «Передача материалов в эксплуатацию»

   1. Движения выполняются в регистр «Расходы»

   2. Суммы берутся из бухгалтерских проводок ДТ 26 - КТ 10.ХХ

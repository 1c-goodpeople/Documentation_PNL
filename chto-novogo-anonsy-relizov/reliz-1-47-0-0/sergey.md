## БП

1. В документе «Платежное поручение» добавлено создание записи платежного календаря из формы документа

2. При создании платежного календаря из документов, в которых отсутствует банковский счет, будет подставлен банковский счет организации по умолчанию

3. В подсистему добавлен новый документ «Премия»

## УНФ

1. Добавлено создание кошелька на основании документа «Заказ покупателя». При этом фактическая сумма оплаты заказа встанет как сумма по кошельку

2. По документу Заказ покупателя добавлена возможность одновременного создания нескольких документов «Операция по кошельку»

3. Добавлен базовый баланс по бухгалтерскому учету

### Соответствие счетов и статей ДДС

1. Добавлено соответствие счетов и статей ДДС / проектов / разделов / доп аналитики

2. Работает следующим образом:

   1. Если в документе указан счет учета (План счетов «Управленческий»), то, если будут найдены данные в соответствии, то в движения по регистрам P&L будут установлены эти данные иначе как при стандартном поведении (из шапки документа)

3. На данный момент реализовано только для документа «Начисление зарплаты УНФ»
